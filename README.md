# Sistema de grids
Este é um simples sistema de grids.

Para usar esse sistema, você deve baixar e colocá-lo no seu HTML.

Assim:

    <link rel="stylesheet" href="grid.css">
    
## Exemplo de uso

Para um exemplo simples

    <div class="container">
      <div class="column-6">...</div>
      <div class="column-6">...</div>
    </div>
